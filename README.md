waht
WordPress a.h Theme

===========
DESCRIPTION
===========

waht (WordPress a.h Theme) is a blank WordPress theme built using HTML5, CSS3 and jQuery.
It aims to be responsive and easily customizable.

Documentation:
    https://github.com/Othella/waht/wiki
